# ⚙️ Stack de Tecnologia

## Backend

### Framework
- **FastAPI** — Framework web assíncrono, rápido, moderna documentação automática
- **Python 3.10+** — Linguagem principal

### ORM e Banco de Dados
- **SQLAlchemy 2.0** — ORM robusta
- **Alembic** — Migrações de banco de dados
- **PostgreSQL 14+** — Banco de dados relacional
- **psycopg2-binary** — Driver PostgreSQL

### Validação e Schemas
- **Pydantic v2** — Validação de dados
- **Pydantic Settings** — Variáveis de ambiente

### Autenticação
- **python-jose** — JWT tokens
- **passlib** — Hashing de senhas
- **bcrypt** — Algoritmo de criptografia

### Testes
- **pytest** — Framework de testes
- **pytest-cov** — Coverage de testes
- **pytest-asyncio** — Testes assíncronos
- **pytest-mock** — Mocking

### Hardware e IoT
- **pyserial** — Comunicação serial (balança)
- **opencv-python** — Visão computacional
- **ultralytics** — YOLO v8
- **numpy** — Computação numérica

### Web Scraping
- **requests** — HTTP cliente
- **beautifulsoup4** — Parser HTML
- **lxml** — Parser XML

### Utilities
- **python-dotenv** — Carregamento de `.env`
- **python-multipart** — Upload de arquivos
- **redis** — Cache e fila (futuro)
- **celery** — Tarefas assíncronas (futuro)
